## Troubleshooting

### Common Issues

1. **CORS Errors**
   - Ensure backend `CORS_ORIGINS` includes the Vercel deployment URL
   - Check that credentials are being sent with requests

2. **Environment Variables Not Applied**
   - Redeploy after adding environment variables
   - Use `vercel --prod --force` to force rebuild

3. **Build Failures**
   - Check TypeScript errors locally with `npm run build`
   - Ensure all dependencies are in `package.json`
